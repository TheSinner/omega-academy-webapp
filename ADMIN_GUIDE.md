## 1. 로그인하기

### 관리자 로그인
1. 웹앱 주소 접속
2. **전화번호** 입력 (예: 01012345678)
3. **비밀번호** 입력 (생년월일 6자리)
4. **자동 로그인** 체크 (선택사항)
5. **로그인** 버튼 클릭

### 로그인 후 화면
- 상단에 **관리자** 배지 표시
- **공지사항**, **질문게시판**, **관리자** 탭 표시

---

## 2. 공지사항 관리

### 공지사항 작성하기

1. **공지사항** 탭 클릭
2. **공지 작성** 버튼 클릭
3. 정보 입력:
   - **제목**: 공지사항 제목 입력
   - **내용**: 공지사항 내용 입력 (여러 줄 가능)
   - **이미지**: 필요시 이미지 첨부 (선택사항)
4. **작성하기** 버튼 클릭

**작성 팁:**
- 제목은 간단명료하게 (예: "10월 정기 시험 안내")
- 내용은 상세하게 작성
- 중요한 정보는 맨 위에 배치
- 이미지는 JPG, PNG 형식 사용

### 공지사항 수정하기

1. 수정할 공지사항 클릭
2. 우측 상단 **연필 아이콘** (수정) 클릭
3. 내용 수정
4. **수정하기** 버튼 클릭

### 공지사항 삭제하기

1. 삭제할 공지사항 클릭
2. 우측 상단 **휴지통 아이콘** (삭제) 클릭
3. 확인 창에서 **확인** 클릭

⚠️ **주의**: 삭제된 공지사항은 복구할 수 없습니다!

---

## 3. 질문게시판 관리

### 게시판 모니터링

관리자는 모든 질문을 확인할 수 있습니다:
- **익명 게시글**: 익명으로 작성되어도 실제 작성자 확인 가능
- **수정 내역**: 수정된 게시글은 수정 횟수 표시
- **삭제된 글**: 삭제 로그에서 확인 가능

### 답변 작성하기

1. **질문게시판** 탭 클릭
2. 답변할 질문 클릭
3. 하단 **답글 작성** 폼에서:
   - 답변 내용 입력
   - 필요시 이미지 첨부
   - **익명** 체크 여부 선택 (체크 해제 권장)
4. **답글 작성** 버튼 클릭

**답변 팁:**
- 명확하고 친절하게 답변
- 관리자 배지가 자동으로 표시됨
- 익명 체크를 해제하여 신뢰도 향상
- 필요시 그림으로 설명

### 부적절한 게시글 삭제

1. 삭제할 게시글/답글/댓글 찾기
2. **휴지통 아이콘** 클릭
3. 확인 창에서 **확인** 클릭

**삭제 시 주의사항:**
- 삭제된 내용은 로그에 자동 저장됨
- 필요시 삭제 로그에서 확인 가능
- 신중하게 결정

---

## 4. 회원 관리

### 관리자 페이지 접속

1. 상단 **관리자** 탭 클릭
2. **회원 관리** 탭 선택

### 새 학생 추가하기

1. **회원 추가** 버튼 클릭
2. 정보 입력:
   - **이름**: 학생 이름
   - **전화번호**: 01012345678 형식
   - **비밀번호**: 생년월일 6자리 (예: 050315)
   - **권한**: 학생 선택
3. **추가** 버튼 클릭

### 조교 추가하기

1. **회원 추가** 버튼 클릭
2. 정보 입력 (학생과 동일)
3. **권한**에서 **조교** 선택
4. **추가** 버튼 클릭

### 회원 정보 수정

1. 회원 목록에서 **연필 아이콘** 클릭
2. 수정할 정보 입력:
   - 이름 변경
   - 권한 변경 (학생 ↔ 조교)
3. **수정** 버튼 클릭

### 비밀번호 재설정

학생이 비밀번호를 잊어버렸을 때:

1. 회원 목록에서 **열쇠 아이콘** 클릭
2. 새 비밀번호 입력 (생년월일 6자리)
3. 학생에게 새 비밀번호 전달

### 회원 삭제

1. 회원 목록에서 **휴지통 아이콘** 클릭
2. 확인 창에서 **확인** 클릭

⚠️ **주의**: 
- 회원을 삭제하면 해당 학생의 모든 질문/답글도 함께 삭제됩니다
- 신중하게 결정하세요

---

## 5. 삭제 로그 확인

### 삭제 로그 보기

1. **관리자** 탭 클릭
2. **삭제 로그** 탭 선택

### 로그 정보

각 로그에는 다음 정보가 포함됩니다:
- **타입**: 질문/답글/댓글
- **삭제자**: 누가 삭제했는지
- **삭제 시간**: 언제 삭제되었는지
- **원본 내용**: "내용 보기" 클릭하여 확인

### 로그 활용

- 부적절한 게시글 추적
- 학생 행동 패턴 파악
- 분쟁 발생 시 증거 자료
- 실수로 삭제한 내용 복구 참고

---

## 6. 권한 체계

### 학생 (Student)
- ✅ 공지사항 읽기
- ✅ 질문 작성/수정/삭제 (본인 것만)
- ✅ 답글/댓글 작성/삭제 (본인 것만)
- ✅ 익명 작성 가능
- ❌ 공지사항 작성 불가
- ❌ 다른 사람 게시글 삭제 불가

### 조교 (Assistant)
- ✅ 학생 권한 모두 포함
- ✅ 답변 시 "조교" 배지 표시
- ❌ 공지사항 작성 불가
- ❌ 회원 관리 불가

### 관리자 (Admin)
- ✅ 모든 기능 사용 가능
- ✅ 공지사항 작성/수정/삭제
- ✅ 모든 게시글 삭제 가능
- ✅ 회원 관리
- ✅ 삭제 로그 확인
- ✅ 익명 게시글의 실제 작성자 확인
- ✅ 답변 시 "관리자" 배지 표시

## 7. 자주 묻는 질문

### Q1: 학생이 비밀번호를 잊어버렸어요
**A**: 관리자 페이지 → 회원 관리 → 해당 학생의 열쇠 아이콘 클릭 → 새 비밀번호 설정

### Q2: 공지사항에 여러 이미지를 넣고 싶어요
**A**: 현재는 공지사항당 1개의 이미지만 첨부 가능합니다. 여러 이미지를 넣으려면 이미지 편집 프로그램으로 합친 후 업로드하세요.

### Q3: 삭제한 공지사항을 복구할 수 있나요?
**A**: 공지사항은 삭제 로그에 저장되지 않습니다. 신중하게 삭제하세요.

### Q4: 질문게시판에서 답변은 어떻게 해야 하나요?
**A**: 질문 상세 페이지 하단의 "답글 작성" 폼을 이용하세요. 관리자 배지가 자동으로 표시됩니다.

### Q5: 조교와 관리자의 차이는 무엇인가요?
**A**: 
- 조교: 질문 답변만 가능
- 관리자: 공지사항 작성, 회원 관리 등 모든 기능 사용 가능

### Q6: 학생이 부적절한 게시글을 올렸어요
**A**: 해당 게시글의 휴지통 아이콘을 클릭하여 삭제하세요. 삭제 로그에 자동 저장됩니다.

### Q7: 새 학생을 추가했는데 로그인이 안 돼요
**A**: 
1. 전화번호 형식이 정확한지 확인 (01012345678)
2. 비밀번호가 6자리 숫자인지 확인
3. 학생에게 정확한 정보를 전달했는지 확인

### Q8: 모바일에서도 관리가 가능한가요?
**A**: 네! 모바일, 태블릿, 컴퓨터 어디서든 접속하여 관리할 수 있습니다.

### Q9: 통계를 볼 수 있나요?
**A**: 관리자 페이지에서 기본 통계 (학생 수, 질문 수, 답글 수)를 확인할 수 있습니다.

### Q10: 여러 명의 관리자를 추가할 수 있나요?
**A**: 네! 회원 추가 시 권한을 "관리자"로 선택하면 됩니다.

## 📱 모바일 관리 팁

- 작성 전 미리보기 권장

### 회원 관리 시
- 가로 모드로 전환하여 표 전체 보기
- 필요시 확대/축소 기능 활용

### 답변 작성 시
- 긴 답변은 컴퓨터 사용 권장
- 간단한 답변은 모바일도 충분

## 🎯 효과적인 운영 팁

### 1. 정기적인 공지사항
- 주 1회 학원 소식 업데이트
- 시험 일정 미리 공지
- 휴원일 사전 안내

### 2. 빠른 답변
- 질문에 신속하게 답변
- 조교 활용으로 답변 속도 향상
- 자주 묻는 질문은 공지사항으로 정리

### 3. 학생 참여 유도
- 좋은 질문에 칭찬 댓글
- 익명 기능으로 부담 감소
- 학생끼리 서로 돕도록 유도

### 4. 정기적인 점검
- 주 1회 삭제 로그 확인
- 부적절한 게시글 모니터링
- 회원 목록 업데이트

### 5. 백업
- 중요한 공지사항은 별도 저장
- 학생 명단 엑셀로 백업
- 정기적인 데이터 확인

## 🔧 문제 해결

### 이미지가 업로드되지 않아요
1. 이미지 크기 확인 (10MB 이하 권장)
2. 이미지 형식 확인 (JPG, PNG)
3. 인터넷 연결 확인

### 회원 추가가 안 돼요
1. 이미 등록된 전화번호인지 확인
2. 전화번호 형식 확인 (01012345678)
3. 비밀번호 6자리 확인

### 로그인이 안 돼요
1. 전화번호 정확히 입력 (하이픈 없이)
2. 비밀번호 정확히 입력 (6자리 숫자)
3. 브라우저 쿠키 설정 확인
